# Release Notes (BG)

## Ъпгрейд на SIMA: UI и Операции

- **Оперативният център е изцяло преработен** в по-практичен control панел с по-чист layout, по-добра четимост и mobile-first поведение.
- **Постоянно запазване за регистрирани потребители** е активно (local + API + Supabase sync), така че задачи и бележки се пазят надеждно.
- **Добавен е RAG асистент в Операции**, включително:
  - ръчни RAG препоръки,
  - автоматичен **Дневен RAG бриф** (веднъж на ден),
  - ръчно пускане на бриф,
  - история на брифовете + бързо отваряне в бележките.
- **Интеграция с Field Watch**:
  - връзка към `/api/fields`,
  - KPI картите вече ползват реални данни (брой полета и обща площ),
  - бърза навигация между Операции и Field Watch.
- **Вързана е риск логика** от профила на фермера:
  - KPI за критични рискове,
  - показване на най-важния текущ риск в Операции.
- **Сезонният календар е свързан с RAG**:
  - AI генерира месечен план за избрана култура/месец,
  - AI задачите се визуализират директно в месечните карти.
- **Визията на календара е подредена**:
  - винаги 12 месеца,
  - при месец без дейности се вижда само сезонна снимка,
  - сезонни fallback визии (напр. януари = зимна сцена),
  - по-добро фокусиране и изрязване на изображенията.
- **Началната страница е опростена**:
  - премахнати са промо блоковете, остава само заглавието `SIMA`.
- **Мобилна навигация/брандинг доизпипани**:
  - центрирано заглавие `SIMA` в горната лента.
- **Финален UX polish**:
  - по-добри отстояния, еднакви височини на KPI карти, по-плавни интеракции,
  - фиксиран контраст в белите полета,
  - добавен е фин skeleton loading ефект за RAG отговори.
